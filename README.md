# DevTec_GameOfLife
How to work with this program:
 First go into build directory and type:
 cmake..

 Then choose what you want to do:
 
  To start the game:
    ./main
  
  To create documentatio:
    make doc
   
  To compress all the source code:
    make dist
  
  To launch cunit tests:
    make test && ./test
